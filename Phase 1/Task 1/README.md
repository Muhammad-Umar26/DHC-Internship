# Task 1 - Iris Dataset EDA

## Task Objective
- Load the Iris dataset with pandas.
- Inspect the dataset shape, column names, first rows, summary statistics, and data types.
- Visualize feature distributions with scatter plots, histograms, and box plots.

## Dataset Used
- Iris dataset from scikit-learn.
- 150 samples with 4 numeric features: sepal length, sepal width, petal length, and petal width.
- 3 target classes: setosa, versicolor, and virginica.

## Models Applied
- No predictive model was trained.
- This task focuses on exploratory data analysis and visualization.

## Key Results and Findings
- The dataset is small, clean, and balanced across the 3 species.
- Petal measurements show the clearest separation between classes, especially for setosa.
- Histograms and box plots help reveal the spread of each feature and highlight class differences.
- Scatter plots make the clustering pattern between species easy to see.
